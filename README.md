This repository contains code and results for a study on the spectral relationship between graph matrices--when used as system matrices--and the corresponding controllability Gramians. The code is provided in jupyter notebooks and the final results were generated using the `verify_multiple_graph_edges_toggling.ipynb` notebook. The results are stored in .csv files at the top level and can be plotted using the `plot_results.py` script. The commands for generating the figures used in the paper are provided in the `figures/figure_generation_commands.sh` script.
